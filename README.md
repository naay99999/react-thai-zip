# react-thaizip

CLI scaffold tool for adding a Thai address autocomplete React component powered by [`thaizip`](https://www.npmjs.com/package/thaizip).

```bash
npx react-thaizip add autocomplete
```

The command detects your React or Next.js project structure, installs `thaizip`, and writes `ThaiAddressAutocomplete.tsx` into the best component directory.
